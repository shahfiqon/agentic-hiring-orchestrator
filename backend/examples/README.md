# Examples

This directory contains example scripts and sample data for the Agentic Hiring Orchestrator.

## Contents

- `run_pipeline.py` - Pipeline execution example (coming soon)
- `sample_data/` - Sample input data for testing the evaluation workflow

## Usage

Example scripts demonstrate how to:
- Run the evaluation pipeline end-to-end
- Submit candidate applications programmatically
- Work with custom rubrics and job descriptions
- Process evaluation results

Check back as examples are added during development.
